在HTML中使用JS的注意事项

发表于 2016-09-13   |   分类于 Tech   |     |   view.

把脚本放在head中，脚本加载导致的延迟可通过一下方式解决：
解决1： js引用放在元素中页面内容后
解决2：HTML4.01定义了defer属性

脚本会延迟到整个页面解析完毕后在运行。同时，defer也保证第一个延迟脚本先于第二个延迟脚本
此外，script还有一个async属性，表示脚本不必等待其他脚本，也不必阻塞文档的呈现。不能保证异步脚本按照它们在页面中出现的顺序执行

使用外部文件来包含js代码的优点：可维护性，可缓存，适应未来。

文档模式：通过doctype来实现切换的。标准模式，混杂模式。主要会影响css内容的呈现，也会影响js的解析。如果在文档开始时，没有文档类型声明，会开启混杂模式。
